Для чего

Для автоматического составления посменного графика работ на месяц.

Для кого

Для управленцев магазинов, сервисов, клиник, ресторанов с большим количеством сотрудников.

Как работает

Вводите условия и получаете готовый план на месяц.

Как пользоваться

1

Укажите требования

Для каждого дня укажите, сколько человек требуется. Отметьте дни, когда сотрудник точно работает или не работает.

2

Получите план

Инструмент автоматически распределит выходные и рабочие дни равномерно между сотрудниками. При необходимости можно подправить план вручную.

3

Сохраните и поделитесь

План можно пересчитывать и сохранять несколько раз. У каждого плана есть секретная ссылка для коллег. Редактировать может только автор.

Попробовать можно прямо здесь. Но чтобы сохранить план, понадобится зарегистрироваться.
